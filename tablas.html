Tablas

Las tablas en las páginas web son necesarias ya que algunos datos las necesitan y facilitan la organización y visualización de estos algunos ejemplos pueden ser precios de acciones, puntajes deportivos, datos de facturas y más.

Crear una tabla

Antes de mostrar datos, primero debe crear la tabla que contendrá los datos utilizando el elemento table. Este elemento contendrá todos los datos tabulares que planea mostrar.

Columnas

Las filas no son suficientes para agregar datos a una tabla. Cada elemento de celda también debe ser definido. En HTML, puede agregar datos mediante la tabla de datos elemento: td.

73 81
En el ejemplo anterior, se ingresaron dos puntos de datos ( 73 y 81) en la fila que existe. Al agregar dos puntos de datos, creamos dos celdas de datos. Si la tabla se mostrara en el navegador, mostraría una tabla con una fila y dos columnas.

Encabezados

Para añadir títulos a las filas y columnas, se puede utilizar el encabezado de tabla elemento: th. El elemento de encabezado de tabla se utiliza como un elemento de datos de tabla, excepto con un título relevante.

Saturday Sunday
Temperature 73 81
¿Qué pasó en el código de arriba? Primero, se agregó una nueva fila para contener los tres encabezados: un encabezado en blanco, un Saturday encabezado y un Sunday encabezado. El encabezado en blanco crea la celda de tabla adicional necesaria para alinear los encabezados de la tabla correctamente sobre los datos a los que corresponden. En la segunda fila, uno encabezado de la tabla se añadió como un título de fila: Temperature. Tenga en cuenta, también, el uso del atributo scope, que puede tomar uno de dos valores:
  • row, - este valor deja claro que el encabezado es para una fila.
  • col, - este valor deja claro que el encabezado es para una columna.
