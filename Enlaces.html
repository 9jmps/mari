Enlaces en una nueva ventana

alguna vez clic en un enlace y observó que la página web resultante se abría en una nueva ventana del navegador, Si es así, puedes agradecer el atributo adel elemento target. El atributo especifica cómo debe abrirse un enlace. Es posible que uno o más enlaces en su página web se vinculen a un sitio web completamente diferente. En ese caso, es posible que desee que los usuarios lean el sitio web vinculado, pero espero que vuelvan a su página web. Esto es exactamente cuando el targetatributo es útil! Para que un enlace se abra en una nueva ventana, el targetatributo requiere un valor de _blank. El atributo target se puede agregar directamente a la etiqueta de apertura del elemento de anclaje, al igual que el atributo href.

En el ejemplo anterior, configurar el atributo target para que "_blank" indique al navegador que abra la página de Wikipedia relevante en una nueva ventana. En este ejercicio, hemos utilizado la terminología "abrir en una nueva ventana". Es probable que esté utilizando un navegador moderno que abre sitios web en nuevas pestañas, en lugar de ventanas nuevas. Antes de la aparición de los navegadores con pestañas, se tenían que abrir ventanas de navegador adicionales para ver más sitios web. El target="_blank" atributo, cuando se usa en los navegadores modernos, abrirá nuevos sitios web en una nueva pestaña.

Enlace en una página relativa

 

 Al crear sitios web estáticos de varias páginas, los desarrolladores web a menudo almacenan archivos HTML en el directorio raíz o una carpeta principal donde se almacenan todos los archivos del proyecto. A medida que aumenta el tamaño de los proyectos que creas, puedes usar carpetas adicionales dentro de la carpeta principal del proyecto para organizar tu código. project-folder/ |—— about.html |—— contact.html |—— index.html El ejemplo anterior muestra tres archivos diferentes: about.html , contact.html e index.html en una carpeta. Los archivos HTML a menudo se almacenan en la misma carpeta, como se muestra en el ejemplo anterior. Si el navegador muestra actualmente index.html , también sabe que about.html y contact.html están en la misma carpeta. Debido a que los archivos se almacenan en la misma carpeta, podemos vincular páginas web mediante una ruta relativa .

En este ejemplo, la etiqueta strong se usa con una ruta relativa para vincular desde el archivo HTML actual al  archivocontact.html en la misma carpeta. En la página web,  aparecerá Contact como un enlace. Una ruta relativa es un nombre de archivo que muestra la ruta a un archivo local (un archivo en el mismo sitio web, como ./index.html) frente a una ruta absoluta (una URL completa, como la https://www.codecademy.com/learn/learn-htmlque se almacena en una carpeta diferente). El ./en ./index.htmlcuenta el navegador para buscar el archivo en la carpeta actual.
