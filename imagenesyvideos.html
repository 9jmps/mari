Imagenes y Video


Imagenes

La etiqueta img le permite agregar una imagen a una página web. La mayoría de los elementos requieren etiquetas de apertura y cierre, pero la etiqueta img es una etiqueta de cierre automático . Tenga en cuenta que el final de la etiqueta tiene una barra diagonal /. Las etiquetas de cierre automático pueden incluir u omitir la barra diagonal final, ambas se representarán correctamente.

La etiqueta img tiene un atributo requerido llamado src. El atributo src se debe establecer en el origen de la imagen o en la ubicación de la imagen. En este caso, el valor de src debe ser el localizador uniforme de recursos (URL) de la imagen. Una URL es la dirección web o la dirección local donde se almacena un archivo.

Alts de la imagen

Parte de ser un desarrollador web excepcional es hacer que su sitio sea accesible para usuarios de todos los orígenes. Para que la web sea más inclusiva, debemos considerar qué sucede cuando las tecnologías de asistencia, como los lectores de pantalla, encuentran etiquetas de imagen.
El atributo alt, que significa texto alternativo, da sentido a las imágenes en nuestros sitios. El atributo alt se puede agregar a la etiqueta de imagen al igual que el srcatributo. El valor de alt debe ser una descripción de la imagen.


El atributo alt también sirve para los siguientes propósitos:
  • Si una imagen no se carga en una página web, un usuario puede pasar el mouse sobre el área originalmente diseñada para la imagen y leer una breve descripción de la imagen. Esto es posible gracias a la descripción que proporciona en el alt atributo.
  • Los usuarios con discapacidades visuales a menudo navegan por la web con la ayuda del software de lectura de pantalla. Cuando incluye el atributo alt el software de lectura de pantalla puede leer en voz alta la descripción de la imagen al usuario con discapacidad visual.
  • alt también desempeña un papel en la optimización de motores de búsqueda (SEO), porque los motores de búsqueda no pueden "ver" las imágenes en los sitios web mientras rastrean internet. Tener atributosalt  descriptivos puede mejorar la clasificación de su sitio.
Si la imagen en la página web no es una que transmita información significativa a un usuario (con discapacidad visual o de otro tipo), el atributo alt debe dejarse vacío.

Videos

Además de las imágenes, HTML también admite la visualización de videos. Al igual que la etiqueta img, la etiqueta video requiere un atributo src con un enlace a la fuente de video.  Sin embargo, a diferencia de la etiqueta de img, el elemento video requiere una apertura y una etiqueta de cierre.

En este ejemplo, la fuente de video ( src) es myVideo.mp4La fuente puede ser un archivo de video alojado junto a su página web, o una URL que apunta a un archivo de video alojado en otra página web. Después del atributo src, los atributos widthy height se utilizan para establecer el tamaño del video que se muestra en el navegador. El atributo controls le indica al navegador que incluya controles de video básicos: pausar, reproducir y saltar. El texto, "Video no compatible", entre las etiquetas de apertura y cierre de video solo se mostrará si el navegador no puede cargar el video.
